My dot.emacs
============

My .emacs


Installation
------------

### Git

```
$ git clone git@github.com:umi-uyura/dot.emacs.d.git ~/.emacs.d
```

### Cask

```
$ brew install cask
$ cd ~/.emacs.d
$ cask
```

Periodic maintenance
--------------------

### Cask

Update package version.

```
$ cask update
```

Update cask itself.

```
$ cask upgrade
```
